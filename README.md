## 🌌 Project Overview
This repository contains an end-to-end data pipeline designed to clean, transform, and analyze cosmic telemetry. By leveraging Python and data science frameworks, the project uncovers hidden insights within astronomical observations, tracing how stars evolve and visualizing exoplanetary distributions.

###  Tech Stack & Libraries Used
- **Core Language:** Python 🐍
- **Data Manipulation:** Pandas, NumPy
- **Data Visualization:** Matplotlib, Seaborn
- **Business Intelligence:** Power BI (Interactive Dashboards)

### 🚀 Key Features & Implementation
- **Data Pipeline:** Engineered custom Python scripts to clean null values and validate schemas across massive astronomical datasets.
- **Exploratory Data Analysis (EDA):** Conducted statistical analysis on star temperatures, luminosity, and planetary mass distributions.
- **Visual Dashboards:** Designed comprehensive charts to plot the Hertzsprung-Russell (H-R) Diagram and track stellar life cycles effectively.
